# Lux SDK v2.1 – A 5D Storytelling Toolkit


Lux SDK is a fully modular development framework designed to create immersive, emotionally resonant experiences using light, sound, motion, and interaction. It enables creators to build truly human-centered digital experiences for the web, VR, AR, gaming, and beyond.


---


## 🌟 Why Lux?


Unlike traditional SDKs, Lux isn't just a set of tools — it's a language of emotion. It's designed to create **interactive environments that feel alive**, resonating with users through visuals, soundscapes, and responsive motion.
